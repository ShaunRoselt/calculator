# Roselt Calculator

Roselt Calculator is an installable all-in-one calculator app built for everyday math, technical work, and quick problem solving. It is heavily inspired by Windows Calculator, then pushes far past it with broader tools, better flexibility, and a more polished cross-device experience.

<img src="docs/Images/CalculatorScreenshot.png" alt="Calculator Screenshot" width="450px" />

## What It Includes

- Standard mode for quick arithmetic with history and memory close at hand
- Scientific mode for constants, logarithms, factorials, powers, and advanced functions
- Programmer mode for alternate bases, bitwise workflows, and binary inspection
- Graphing mode for plotting expressions and exploring functions visually
- Date calculation tools for finding differences and shifting dates
- Converters for currency, time, length, speed, pressure, data, and more
- Installable app behavior on desktop and mobile with responsive layouts across screen sizes

## Why Roselt Calculator

Roselt Calculator is designed to feel immediate. You can open it in the browser for a quick calculation, keep using it for deeper workflows like graphing or programmer tasks, and install it later if it earns a permanent place in your toolkit.

It keeps the feature set broad without turning the interface into a maze. The goal is a single calculator surface that still feels clear whether you are doing simple math, checking a unit conversion, exploring a graph, or working through technical inputs, while giving you more reach than the stock Windows Calculator workflow.

## Best Fit For

- Students who want one calculator app for classwork, conversions, and graphing
- Developers and technical users who need scientific and programmer tools in the same place
- Anyone who wants a web app that can also behave like a lightweight installed calculator

## Availability

Roselt Calculator runs in the browser and can be installed as a progressive web app. Platform-specific development, packaging, validation, and contribution details live in [CONTRIBUTING.md](CONTRIBUTING.md).
