Modes

Purpose-built tools behind the same familiar chrome.

The design starts with Windows Calculator familiarity, then pushes further with broader workflows, cleaner responsiveness, and a better installable web experience.

StandardCore arithmetic with history and memory support.
ScientificFunctions, constants, and advanced operations in a compact layout.
ProgrammerBitwise tools, alternate bases, and binary inspection for technical work.
GraphingPlot expressions and adjust the viewport without leaving the browser.
DateCalculate ranges and offsets for practical scheduling and planning.
ConverterConvert currency, time, area, speed, pressure, data, and more.

Install and go

Keep the web convenience. Add app-like presence.

Roselt Calculator runs entirely client side, supports installation as a progressive web app, and keeps your working state close at hand whether you open it in a tab or as a standalone window.

Desktop ready Install from Chromium-based browsers for a dedicated windowed experience.
Mobile friendly Responsive layouts adapt to touch screens without reducing the feature set.
Private by default History, memory, and preferences stay in local browser storage.
